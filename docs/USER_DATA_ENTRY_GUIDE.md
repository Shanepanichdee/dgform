# คู่มือการลงทะเบียนบัญชีข้อมูลโดยละเอียด (Comprehensive Data Entry Guide)

คู่มือนี้รวบรวมคำอธิบายทุกช่องสถานะในระบบ **DG Data Repository** เพื่อให้เจ้าหน้าที่ผู้ปฏิบัติงานสามารถกรอกข้อมูลได้ครบถ้วน ถูกต้องตามมาตรฐานธรรมาภิบาลข้อมูล (Data Governance)

---

## ส่วนที่ 0: ข้อมูลพื้นฐานหน่วยงาน
ก่อนเริ่มกรอกส่วนอื่น โปรดระบุข้อมูลผู้ส่งดังนี้:

| ชื่อช่องข้อมูล | คำอธิบาย | ตัวอย่างการกรอก |
| :--- | :--- | :--- |
| **หน่วยงานผู้ส่ง (Submitter Agency)** | ชื่อหน่วยงานที่รับผิดชอบการนำส่งข้อมูลจริง | กรมโรงงานอุตสาหกรรม, กรมการปกครอง |
| **Business Domain** | ขอบเขตธุรกิจหลักที่ชุดข้อมูลนี้สังกัดอยู่ | ประชาชน, งบประมาณและการคลัง |

---

## ส่วนที่ 1: คำศัพท์ทางธุรกิจ (Business Glossary)
ใช้สำหรับอธิบาย "คำเฉพาะ" ที่อาจทำให้คนนอกเข้าใจผิดได้

*   **คำศัพท์ (Term):** ชื่อคำศัพท์ที่ต้องการนิยาม (เช่น "โรงงานจำพวกที่ 3")
*   **ความหมาย (Definition):** อธิบายรายละเอียดของคำนั้น (เช่น "โรงงานที่มีแรงม้าเกิน 50 แรงม้า หรือมีมลพิษที่ต้องควบคุม...")
*   **เจ้าของ/แหล่งอ้างอิง:** ระบุที่มาของนิยามนี้ (เช่น "มาตรา 7 พรบ.โรงงาน พ.ศ. 2535")

## ส่วนที่ 2: คำอธิบายชุดข้อมูล (Metadata)

### 2.1 ข้อมูลทั่วไป (General Info)
| ชื่อช่องข้อมูล | คำอธิบายละเอียด | ตัวอย่างการกรอก |
| :--- | :--- | :--- |
| **ชื่อชุดข้อมูล (Title)** | ชื่อเรียกที่เป็นทางการของชุดข้อมูล | ทะเบียนรายชื่อโรงงานอุตสาหกรรมทั่วประเทศ |
| **คำอธิบาย (Description)** | รายละเอียดว่า "ข้อมูลนี้เกี่ยวกับอะไร" | ข้อมูลรายชื่อโรงงานที่ได้รับใบอนุญาต รง.4 ประกอบด้วยชื่อโรงงาน ที่ตั้ง และประเภทอุตสาหกรรม |
| **วัตถุประสงค์ (Objective)** | บอกเหตุผลว่า "เก็บข้อมูลนี้ไว้ทำไม" | เพื่อใช้ในการติดตาม ตรวจสอบ และกำกับดูแลความปลอดภัยของโรงงานอุตสาหกรรม |
| **คำสำคัญ (Keywords)** | คำสั้นๆ สำหรับใช้ค้นหา (พิมพ์แล้วกด Enter) | โรงงาน, รง.4, อุตสาหกรรม, ใบอนุญาต |

### 2.2 การจัดจำแนกและธรรมาภิบาล (Governance)
*   **กลุ่มหัวข้อ (Topic):** เลือกหมวดหมู่ที่ใกล้เคียงที่สุด (เช่น เศรษฐกิจ, สิ่งแวดล้อม)
*   **หมวดหมู่ข้อมูล (Data Category):**
    *   *Public:* เปิดเผยได้ทั่วไป
    *   *Internal:* ใช้เฉพาะภายในหน่วยงาน
    *   *Personal:* มีข้อมูลที่ระบุตัวตนคนได้
*   **ชั้นความลับ (Confidentiality):** เลือกระดับความสำคัญ (เปิดเผย, ลับ, ลับที่สุด)
*   **ประเภทชุดข้อมูล (Dataset Type):** รูปแแบบเนื้อหา (เช่น ระเบียน (Record) หรือ สถิติ)
*   **บทบาทข้อมูล (Functional Role):** เช่น Master (ข้อมูลหลัก) หรือ Transaction (ข้อมูลรายการที่เกิดขึ้นบ่อย)
*   **รูปแบบ (Structure):** Structured (มีตารางชัดเจน) หรือ Unstructured (เช่น PDF)
*   **สัญญาอนุญาต (License):** เงื่อนไขกฎหมายในการนำไปใช้ (เช่น Open Government License)

### 2.3 ผู้รับผิดชอบและขอบเขต (Owner & Technical)
*   **หน่วยงานเจ้าของข้อมูล:** กรมที่ถือครองกฎหมายหรืออำนาจสูงสุดในข้อมูลนี้
*   **หน่วยงานปรับปรุงข้อมูล:** ฝ่ายหรือกองที่เป็นคนคีย์ข้อมูลหรือดูแลระบบ
*   **อีเมลผู้ติดต่อ:** ช่องทางให้คนอื่นติดต่อมาขอใช้ข้อมูลเพิ่มเติม
*   **แหล่งที่มา:** ข้อมูลนี้ได้มาอย่างไร (เช่น "จากระบบจดทะเบียนออนไลน์", "จากการสำรวจรายเดือน")
*   **รูปแบบไฟล์ (Format):** เช่น CSV, XLSX, API, หรือ PDF
*   **ความถี่การปรับปรุง:** เช่น ปีละ 1 ครั้ง, เดือนละ 1 ครั้ง, หรือ Realtime (ทันที)
*   **ขอบเขตภูมิศาสตร์:** พื้นที่ที่ข้อมูลครอบคลุม (เช่น ทั่วประเทศ, กรุงเทพฯ, หรือ ภาคตะวันออก)
*   **URL เข้าถึงข้อมูล:** ลิงก์ดาวน์โหลดไฟล์จริง (ถ้ามี)

### 2.5 ข้อมูลทางเลือก (Optional Metadata - กดขยายเพื่อกรอก)
*   **วันที่สร้างข้อมูล:** วันที่เริ่มมีชุดข้อมูลนี้เป็นครั้งแรก
*   **วันที่แก้ไขล่าสุด:** วันที่อัปเดตไฟล์ล่าสุด
*   **เงื่อนไขการเข้าถึง:** เช่น "ต้องทำหนังสือขออนุญาตก่อนเข้าใช้งาน"
*   **ผู้สนับสนุน:** หน่วยงานภายนอกที่ช่วยออกงบหรือสนับสนุนข้อมูล
*   **หน่วยย่อยที่สุด (Unit of Analysis):** ระดับที่เล็กที่สุดของข้อมูล (เช่น 1 แถว = 1 โรงงาน หรือ 1 แถว = 1 จังหวัด)

## ส่วนที่ 3: พจนานุกรมข้อมูล (Data Dictionary)
ส่วนที่สำคัญที่สุด: อธิบายความหมายของแต่ละ "คอลัมน์" (Column) ในไฟล์ข้อมูลของคุณ

| ชื่อหัวข้อ | คำอธิบาย | ตัวอย่าง |
| :--- | :--- | :--- |
| **ชื่อฟิลด์** | ชื่อหัวคอลัมน์ในไฟล์จริง | factory_id, factory_name |
| **คำอธิบายฟิลด์** | คอลัมน์นี้เก็บค่าอะไร | เลขทะเบียนโรงงาน (13 หลัก) |
| **ประเภทข้อมูล** | ชนิดของข้อมูล | String (ข้อความ), Number, Date |
| **รูปแบบ** | รูปแบบเฉพาะ | DD/MM/YYYY (สำหรับวันที่) |
| **เงื่อนไขการกรอก** | กฎการเช็คข้อมูล | ห้ามว่าง, ห้ามซ้ำ, ต้องเป็นตัวเลขเท่านั้น |
| **แหล่งที่มา** | ต้นทางของคอลัมน์นี้ | จากฐานข้อมูลทะเบียน รง. |
| **ข้อมูลส่วนบุคคล (Has PII)** | มีข้อมูลระบุตัวคนไหม? | Yes (ถ้าเป็นชื่อคน), No (ถ้าเป็นชื่อบริษัท) |
| **ระดับความลับ** | ความลับเฉพาะฟิลด์นี้ | เปิดเผย, ลับ |
| **การเผยแพร่** | เผยแพร่ให้ใครได้บ้าง | สาธารณะ, เฉพาะในกอง |
| **ความยาวข้อมูล** | จำนวนอักขระที่ยอมให้กรอก | 13, 255 |
| **หมายเหตุ** | ข้อมูลเพิ่มเติมอื่นๆ | เริ่มเก็บตั้งแต่ปี 2560 เป็นต้นมา |

> [!TIP]
> **เคล็ดลับการกรอก:**
> 1. หากคุณมีข้อมูลเดิมอยู่ในไฟล์ CSV สามารถกดปุ่ม **Import** เพื่อลดเวลาการคีย์ข้อมูลได้
> 2. หากหน่วยงานของคุณมีฟิลด์พิเศษที่ระบบไม่มีให้ คุณสามารถกดปุ่ม **"+ เพิ่มฟิลด์ใหม่"** ได้ในทุกหน้า
